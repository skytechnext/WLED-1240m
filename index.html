2. Multi-Controller Architecture

A single QuinLED-Dig-Octa Brainboard can drive ~2,000–3,000 addressable pixels at 40+ FPS. For 24,800 segments across 1,240m, the system is divided into zones, each managed by its own controller.

Key formula:
Controllers needed = Total segments ÷ Pixels per controller
24,800 ÷ 2,500 = ~10 Brainboards

Each Brainboard has 8 outputs × ~300 segments each = 2,400 segments per board, covering ~124m of strip.

How WLED Sync Creates a Seamless Wave

WLED's UDP Sync feature (port 21324) broadcasts the current effect state to all controllers on the same network. Each controller is assigned an offset value so the wave pattern continues seamlessly from one zone to the next, rather than each zone starting the same pattern independently.

Offset calculation:
Offset per zone = (Zone number × Segments per zone) ÷ Total segments × 255

For 10 zones: Zone 1 = offset 0, Zone 2 = offset 26, Zone 3 = offset 51, ... Zone 10 = offset 230.
This creates a continuous wave that flows across the full 1,240m installation.

3. Power Injection Diagram

At 21 W/m and 24V, the current draw is 0.875 A/m. On a 24V strip, voltage drop becomes significant after ~5m. Power must be injected every 5 meters to maintain consistent brightness and color accuracy.

Important: Never daisy-chain power through the LED strip itself over long distances. Always run a parallel heavy-gauge power bus and tap into the strip at each injection point. This prevents the thin PCB traces from carrying excessive current.

4. Data Input Topology

The WS2811 data signal is a one-wire serial protocol at 800 kHz. Each Dig-Octa output sends data to one strip segment. The data signal only needs to connect at the start of each strip run.

Key takeaway: The wave pattern works by dividing 1,240m into 10 zones of ~124m each. Each zone is driven by one Dig-Octa controller with 8 parallel strip outputs. WLED's UDP Sync broadcasts effect timing across all 10 controllers over Wi-Fi, with per-controller offsets creating a continuous wave that flows seamlessly across the entire installation.
